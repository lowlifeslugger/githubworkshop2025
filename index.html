How to Participate

Follow these simple steps to contribute to our workshop project and get your page live!

1

Fork Repository

Click the "Fork" button on our GitHub repository to create your own copy.

2

Create Your Page

Add your HTML file in the /attendees/ folder with your unique content.

3

Update Navigation

Add your name and page link to the "Attendee Pages" dropdown menu.

4

Submit Pull Request

Commit your changes, push to your fork, and create a Pull Request.

5

Go Live!

Once approved and merged, your page will be live on GitHub Pages!
